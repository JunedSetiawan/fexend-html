# Fexend Project

A dashboard project using tailwind css, alpine js

## Setup

1. Clone the repository:

```bash
git clone https://github.com/zera-app/fexend-html.git
```

2. Navigate to the project directory:

```bash
cd fexend-html
```

3. Watch Css File

```bash
npm install && npm run dev
```
